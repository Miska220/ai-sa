Конфигурация окружения для воркшопа

Для успешного прохождения 2-го дня фестиваля используйте следующие настройки таймаута для ваших локальных агентов:

# Настройка для Ollama при работе с тяжелыми моделями
$env:OLLAMA_REQUEST_TIMEOUT = "600s"
ollama serve

Это обеспечит стабильную генерацию кода даже на стандартных системах с 8GB RAM.
