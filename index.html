СИСТЕМА АКТИВНА

Этот портал является техническим расширением основного лендинга на Tilda. Здесь аккумулируются данные по воркшопам, спецификации нейросетей и инструменты для локального развертывания AI-агентов.

Текущая фаза: Подготовка окружения (Pre-alpha).

Программа интенсива

Модуль 1: Визуальный инжиниринг

Изучение стилистики Mobile Legends и Cid Kagenou для создания игровых ассетов через Stable Diffusion.

Модуль 2: Локальные LLM

Настройка Ollama и Cline. Оптимизация под системы с 8GB RAM. Устранение ошибок Request Timeout через системные переменные.

Модуль 3: Автоматизация

Написание Selenium-скриптов для сбора данных и тестирования веб-интерфейсов.
